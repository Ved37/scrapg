DO this before running...
1. go to scrapg/client and run "npm i"
2. go to scrapg/client/server and run "npm i"
3. create a .env file in "..\scrapg\.env" with mongo_url,cloud_name,cloud_api_key,cloud_api_secret
To run the program 
1. Go to scrapg/client folder and run "npm start"
2. Go to scrapg folder and run "npx nodemon server/server"
